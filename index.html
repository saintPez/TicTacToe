Introducción

Tic Tac Toe también conocido como Triqui o Tres en raya, es un juego de estrategia, comúnmente entre dos jugadores X y O, que son marcados en un espacio de un tablero 3x3 alternadamente

Variantes

Las variantes de este juego son infinitas, entre la variedad de tableros, variedad de jugadores y la cantidad de marcas(X o O) para ganar

Ejemplo


Juego 11,10,5

API REST

es una interfaz de programación de aplicaciones (API) de transferencia de estado representacional (REST)

por protocolo HTTP: Protocolo de transferencia de hipertexto

Sockets

es un concepto por el cual dos procesos pueden intercambiar cualquier flujo de datos

por protocolo TCP: Protocolo de control de transmisión. los datos pueden enviarse de forma bidireccional

Encriptacion Salt

es un tipo de encriptacion, usada para encriptar contraseñas

Salt generada automáticamente y se une con la contraseña. no se desencripta, se compara con contraseña encriptada

JWT

es un estándar abierto basado en JSON para la creación de tokens de acceso

Header: información del algoritmo Payload: información a encriptar Sing: firma con una clave secreta

Speech Recognition

es el reconocimiento automático de voz, ya esta implementado en los navegadores
